Miles of Tile Styles — Setup Cheatsheet

Website + Professional Email · 7 Steps · ~30 Minutes

milesoftilestyles.com
Kevin Gardner · Boulder, CO
Total cost: ~$12/year (domain only)
Web: Browser → milesoftilestyles.com → Cloudflare → GitHub Pages (free hosting)
Email: someone@... → kevin@milesoftilestyles.com → Cloudflare → Kevin's Gmail inbox
Reply: Kevin hits reply in Gmail → sends as kevin@milesoftilestyles.com → client never sees personal Gmail
1
Buy the Domain
Go to Domain.com or Namecheap.com. Search for milesoftilestyles.com. Purchase (~$12/yr). Enable domain privacy if offered. Do NOT buy their email add-on.
2
Create Free Cloudflare Account
Go to cloudflare.com → Sign up → Click "Add a Site" → Enter milesoftilestyles.com → Select Free plan. Cloudflare gives you two nameserver addresses. Copy them.
3
Point Domain to Cloudflare
Log back into your registrar (Domain.com/Namecheap). Find Nameservers settings. Replace existing nameservers with the two Cloudflare ones. Save. Wait 15–60 min. Domain shows "Active" in Cloudflare.
4
Set Up Email Forwarding
In Cloudflare → your domain → Email → Email Routing → "Get Started" → let it add DNS records automatically.
  • Go to Routing Rules → Create Address
  • Custom address: kevin
  • Destination: Kevin's Gmail address
  • Save → Check Gmail for verification email → Click verify link
Optional: Enable "Catch-all" so ANY @milesoftilestyles.com email goes to Gmail.
5
Set Up Gmail "Send As" (Reply From Business Address)
5a. Google Account → Security → App Passwords → Select "Mail" → Generate → copy the 16-character password.

5b. Gmail → ⚙ Settings → Accounts and Import → "Send mail as" → "Add another email address"
  • Name: Kevin Gardner
  • Email: kevin@milesoftilestyles.com
  • SMTP Server: smtp.gmail.com · Port: 587 · TLS
  • Username: Kevin's Gmail address · Password: the App Password from 5a
  • Click "Add Account" → enter the confirmation code from the email that arrives
Make it default: In Send mail as, click "make default" next to the business address.
6
Connect Website to Domain
6a. GitHub repo → Settings → Pages → Custom domain: milesoftilestyles.com → Save → Check "Enforce HTTPS"

6b. In Cloudflare DNS, add these records:
TypeNameContentProxy
A@185.199.108.153DNS only
A@185.199.109.153DNS only
A@185.199.110.153DNS only
A@185.199.111.153DNS only
CNAMEwwwgerald1950ggg-ai.github.ioDNS only

6c. Update the SPF record to authorize Gmail sending:
TypeNameContent
TXT@v=spf1 include:_spf.mx.cloudflare.net include:_spf.google.com ~all
TXT_dmarcv=DMARC1; p=none; rua=mailto:kevin@milesoftilestyles.com
7
Test Everything
milesoftilestyles.com loads in browser
www.milesoftilestyles.com also works
Padlock icon shows (HTTPS)
Email to kevin@... arrives in Gmail
Reply from Gmail shows business address
New compose → "From" has business option
